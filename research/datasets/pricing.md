## 1. SaaS Tiered Pricing

### Pattern S1: 3-Tier Standard (Starter → Growth → Enterprise)
```
Starter (SAR 199/mo)      Growth (SAR 599/mo)      Enterprise (SAR 1,999/mo)
- 1 user                   - 5 users                - Unlimited users
- Basic features           - Advanced features       - All features
- Email support            - Priority support        - 24/7 dedicated support
- 1,000 transactions       - 10,000 transactions     - Unlimited transactions
- Standard reporting       - Advanced analytics      - Custom reporting + BI
```

### Arabic Example
```
المبتدئ (199 ريال/شهر)     النمو (599 ريال/شهر)      المؤسسات (1,999 ريال/شهر)
- مستخدم واحد               - 5 مستخدمين              - مستخدمين غير محدودين
- ميزات أساسية              - ميزات متقدمة             - جميع الميزات
- دعم عبر البريد            - دعم ذو أولوية            - دعم مخصص 24/7
- 1,000 معاملة              - 10,000 معاملة            - معاملات غير محدودة
```

### Key Elements
| Element | Importance | Implementation |
|---|---|---|
| Most popular badge | Critical | Highlight middle tier in accent color |
| SAR prices | Mandatory | Local currency + USD toggle |
| Annual discount | Common | "Save 20% annually" badge |
| Free trial CTA | Expected | "Start Free Trial" on all tiers |
| Feature comparison | Important | Expandable comparison table |
| VAT mention | Mandatory | "+15% VAT" clearly stated |

### Pattern S2: Feature-Gated 3-Tier
```
Free (SAR 0)     Pro (SAR 299/mo)     Business (SAR 999/mo)
Limited          Unlimited            Unlimited + API
1 project        10 projects          Unlimited projects
Community         Email + Chat        Priority + Phone
```

### Pattern S3: Usage-Based Tiered
```
Light (SAR 149/mo)    Standard (SAR 499/mo)    Pro (SAR 999/mo)
500 contacts          5,000 contacts          50,000 contacts
Basic automation      Workflow automation     AI-powered automation
                      +SAR 0.10/extra contact
```

## 10. Common Pricing Page Questions in MENA

| Question | Positioning | Answer Strategy |
|---|---|---|
| "Is this in SAR or USD?" | Top of card | Always show SAR, toggle for USD |
| "Is VAT included?" | Below price | Show +15% VAT separately or total |
| "Can I pay monthly?" | Near CTA | Toggle monthly/annual clearly |
| "What if I need more?" | Enterprise tier | "Custom pricing available" |
| "Is there a setup fee?" | FAQ section | Be transparent — yes/no |
| "Do I need a credit card?" | Below trial CTA | "No credit card required" |
| "Can I cancel anytime?" | Near checkout | "Cancel anytime, no questions" |
| "Is data migration included?" | Enterprise plan | Feature listed or add-on |
| "Do you serve government?" | Enterprise tier | Special government plan |
| "Is there a free trial?" | Hero section | "14-day free trial" badge |

### Most Effective Pricing Page Patterns in MENA
1. **3-tier card layout** with "Most Popular" badge — highest conversion
2. **Annual/monthly toggle** — annual preferred (40%+ discount shown)
3. **Money-back guarantee** — 30-day guarantee adds 15% to conversion
4. **Saudi data residency note** — "Your data stays in KSA" as bullet point
5. **Free trial without credit card** — expectation, not differentiator
6. **Consultative enterprise CTA** — "Talk to our team" > "Buy Enterprise"
7. **Feature comparison** — visual checkmarks over text descriptions
8. **Case study thumbnails** — on enterprise tier card
