## 1. SaaS Hero Patterns

### Pattern S1: "The Platform for X" Structure
```
Headline:  [Name] — The leading [Platform Type] in [Region]
Subhead:   [Value prop sentence] — used by [Number] [Client type]
CTA:       Start Free Trial | Book a Demo
Trust:     Logo bar of clients + [Number] users + [Rating]
```

| Element | EN Example | AR Example |
|---|---|---|
| Headline | Salla — The Leading Saudi E-commerce Platform | سلة — منصة التجارة الإلكترونية السعودية الرائدة |
| Subhead | Start, manage, and grow your online store in minutes. 70,000+ merchants trust us. | ابدأ وأدر وأنمِ متجرك الإلكتروني في دقائق. أكثر من 70,000 تاجر يثقون بنا. |
| CTA | Start Free Trial | ابدأ النسخة التجريبية المجانية |
| Trust | 70K+ merchants · 4.7★ · Saudi-owned | 70K+ تاجر · 4.7★ · مملوك سعودياً |
| Layout | Centered headline + search bar + category grid below | |
